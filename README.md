# Análise de Dados com Python — Spotify e Netflix

Este repositório reúne dois projetos de análise de dados desenvolvidos em Python, utilizando datasets do Spotify e da Netflix. O objetivo foi praticar limpeza, transformação, consulta e visualização de dados com diferentes ferramentas do ecossistema Python.

## Projetos
### Spotify

Análise de dados musicais utilizando **PySpark** e **Spark SQL**.

Principais atividades:
- leitura e estruturação de dados em um Spark DataFrame;
- remoção de valores nulos e registros duplicados;
- identificação das músicas mais populares;
- filtragem de músicas e álbuns por artista;
- cálculo da duração média de músicas com alta popularidade;
- exploração dos gêneros presentes no dataset.

### Netflix

Análise do catálogo da Netflix utilizando **Pandas**, **NumPy** e bibliotecas de visualização.

Principais atividades:
- tratamento de valores ausentes;
- preenchimento de dados categóricos com a moda;
- análise da distribuição entre filmes e séries;
- identificação dos países com mais títulos;
- análise de lançamentos por ano;
- levantamento dos gêneros e diretores mais frequentes;
- análise da distribuição da duração dos filmes;
- criação de gráficos com Matplotlib, Seaborn e Plotly.

## Tecnologias utilizadas

- Python
- PySpark
- Spark SQL
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Jupyter Notebook / Google Colab
